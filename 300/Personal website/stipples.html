a procedural drawing tool

Stipples is an interactive sketch created in Processing which allows the user to create procedurally generated, abstract graphics with a group of moving dots. The initial goal of the project was to create a looping animation with circles moving uniformly and changing in size, color. etc. With some tinkering in the code, the finished project morphed into a drawing tool with five integrated color palettes and user control over size, speed, opacity, and color of the dots.

The controls are as follows:

  • # keys 1-5: cycle through color palettes
  • A: change alpha
  • R: change radius of dots
  • S: change speed
  • Left mouse click: capture current frame
